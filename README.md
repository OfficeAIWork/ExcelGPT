## 软件简介

ExcelGPT 是一款智能表格处理插件，覆盖公式解释与生成、数据处理与图表、文本润色改写、中英文翻译、智能问答等全套表格处理功能，降低表格工作的学习成本与门槛，让您把时间精力集中在创造价值上。

| 核心能力 | 说明 |
| :--- | :--- |
| **公式解释与生成** | 自动解析公式含义与运算步骤，根据自然语言描述智能生成公式 |
| **数据处理与图表** | 根据需求描述生成或编辑数据，自动识别数据关系生成可视化图表 |
| **文本润色与翻译** | 疏通润色单元格文本、中英文互译，提升文案质量 |
| **AI 公式函数** | 内置 `AI.Ask`、`AI.Execute`、`AI.Extract` 三大公式，在单元格中直接调用 AI 能力 |
| **办公智能体** | 多模态智能助手，支持检索文件、生成图表/思维导图/图片、辅助编辑文档 |

> **适配环境：** Windows / macOS · Office 2012+ · WPS 最新版

## 软件使用

### 公式解释

自动解析当前选定单元格中公式的函数用途和运算步骤，帮助您轻松看懂公式。如果公式有误，会自动检查并给出正确公式建议，结果以批注形式备注在单元格中。

![公式解释](./Images/5.公式解释.png)

### 公式生成

根据问题或需求描述，自动在当前选中的单元格中生成对应公式。

![公式生成](./Images/6.公式生成.png)

### 数据处理

根据需求描述对数据进行生成或编辑处理，无需手动编写复杂公式。

![数据处理](./Images/7.数据处理.png)

### 生成图表

自动识别数据的关联关系，将选中的数据生成指定的图表形式。

![图表生成](./Images/8.图表生成.png)

### 思维导图

> [!WARNING]
> 仅支持 Excel，WPS 表格暂不支持

根据选定的单元格内容生成横向或纵向的思维导图。

![思维导图](./Images/9.思维导图.png)

### 疏通润色

选择需要改写的单元格，点击【疏通润色】，自动改写润色当前单元格文本内容。

### 中英互译

将选中单元格中的内容进行中英文转换。

### OfficeAI 助理

OfficeAI 助理是一个完全免费的智能对话机器人，擅长通过自然语言连续对话处理文本及数据，包括知识问答、语言翻译、生成各类文案、总结摘要、生成测试数据、数据分析等。遇到不会用的 Excel / WPS 功能，可以直接向助理提问。

> [!TIP]

### AI 公式

> [!WARNING]
> 仅支持 Excel，WPS 表格暂不支持

#### AI.Ask(prompt) — 提问公式

向 AI 提问并获取回答。

| 参数 | 说明 |
| :--- | :--- |
| `prompt` | 问题内容（文本或单元格引用） |

**调用方式：**

```
=AI.Ask("如何使用Excel制作工资表")    ← 传入文本
=AI.Ask(A1)                           ← 传入单元格
```

![AI.Ask 示例](./Images/13.ASK函数.png)

#### AI.Execute(content, operation) — 执行命令公式

对数据执行指定操作并返回处理结果。

| 参数 | 说明 |
| :--- | :--- |
| `content` | 需要处理的数据 |
| `operation` | 对数据执行的操作 |

**调用方式：**

```
=AI.Execute("广东", "省会")            ← 传入文本
=AI.Execute(A1, B1)                    ← 传入单元格
=AI.Execute("广东", B1)                ← 混合传入
=AI.Execute(A1, "省会")                ← 混合传入
```

![AI.Execute 示例](./Images/14.Execute函数.png)

#### AI.Extract(content, operation) — 内容提取公式

按规则从数据中提取信息，多个结果以分号 `;` 分隔。

| 参数 | 说明 |
| :--- | :--- |
| `content` | 需要处理的数据 |
| `operation` | 提取数据的规则 |

**调用方式：**

```
=AI.Extract("王总在珠江新城开会", "提取其中的地名")    ← 传入文本
=AI.Extract(A1, B1)                                   ← 传入单元格
=AI.Extract(A1, "提取其中的人名")                      ← 混合传入
```

![AI.Extract 示例](./Images/15.Extract函数.png)
